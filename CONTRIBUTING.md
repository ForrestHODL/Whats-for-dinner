# Contributing

Thanks for your interest in Meal Prep Planner!

## Security

- Do **not** commit `.env` or Supabase **service_role** keys.
- Report security issues privately to the repository owner rather than opening a public issue with sensitive details.

## Development

```bash
npm install
cp .env.example .env
npm run dev
```

## Pull requests

1. Fork the repo and create a branch.
2. Keep changes focused.
3. Ensure `npm run build` passes.
